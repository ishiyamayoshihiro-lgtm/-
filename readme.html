📚 学習アプリ

使い方ガイド & リンク集

2つの学習アプリを提供しています

Google認証で安全にログインし、テスト結果を自動的にスプレッドシートに記録します。
PC・iPadどちらでもご利用いただけます。

📐三角比小テスト

三角比の基礎を確認するテストアプリです。値を求める問題と角度を求める問題の2つのモードがあります。

  • 値を求める: 27問
  • 角度を求める: 21問
  • KaTeX表記で美しい数式表示
  • 自動採点・結果記録
📝 テスト開始 📖 詳細を見る

🔢百ます計算

計算力を鍛える百ます計算アプリです。10×10のマス目に素早く正確に答えを入力します。

  • 足し算モード: 100問
  • 0-9の数字が重複なし
  • タイマー機能搭載
  • 自動採点・結果記録
📝 テスト開始 📖 詳細を見る

共通機能

  • Google OAuth 2.0認証 - 安全なログイン
  • 自動採点機能 - 提出後すぐに結果表示
  • スプレッドシート連携 - テスト結果を自動記録
  • 送信リトライ機能 - 最大3回自動リトライ + 手動再送信
  • レスポンシブデザイン - PC・iPad対応
